Cascading Style Sheets

CSS

Que siginifie CSS ?

CSS (Cascading Style Sheets ou Feuilles
de style en cascade en français)
est un langage déclaratif utilisé pour
décrire la présentation de pages web dans le
navigateur. Le navigateur applique
les déclarations de style CSS aux éléments
concernés pour les mettre en forme.
Une déclaration de style contient des
propriétés et leurs valeurs et détermine
l'apparence d'un ou plusieurs éléments
de la page.

Fonctionnement

Comment ca marche ?

CSS est l'une des trois technologies qui
constituent le noyau du web,
avec HTML et JavaScript.
Ce langage est principalement utilisé pour
appliquer un style aux éléments HTML,
mais peut aussi être utilisé avec d'autres
langages de balisage tels que SVG ou encore XML.
Une règle CSS est un ensemble de propriétés
associé à un sélecteur.
